# Student CGPA API

## Objective
REST API built using Express.js to manage student CGPA records stored in an in-memory JSON array.

## Implemented Routes

GET /students → Get all students  
GET /students/topper → Get student with highest CGPA  
GET /students/average → Get average CGPA  
GET /students/count → Get total students  
GET /students/:id → Get student by ID  
GET /students/branch/:branchName → Get students by branch
